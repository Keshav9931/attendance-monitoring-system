# 🎯 Face Recognition Based Attendance Monitoring System

## 📌 Overview
This project is an AI-based Attendance Monitoring System that uses Face Detection and Recognition to automatically mark attendance. It eliminates the need for manual attendance and improves accuracy and efficiency.

---

## 🚀 Features
- 👤 Face Detection using OpenCV
- 🧠 Face Recognition using LBPH Algorithm
- 📸 Capture and store face images
- 📝 Automatic attendance marking
- 📊 Attendance stored in CSV files
- 🖥️ User-friendly GUI using Tkinter

---

## 🛠️ Tech Stack
- Python
- OpenCV (cv2)
- NumPy
- Pandas
- Pillow (PIL)
- Tkinter (GUI)
